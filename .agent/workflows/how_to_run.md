---
description: How to run the application locally
---
1. Open your terminal.
2. Ensure you are in the project root directory (`d:\KULIAH\MAGANG\INALUM\PROJECT\DEV`).
3. Install dependencies (if you haven't already):
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. The application will start and usually be accessible at `http://localhost:5173` or `http://localhost:5181` (check the terminal output).
